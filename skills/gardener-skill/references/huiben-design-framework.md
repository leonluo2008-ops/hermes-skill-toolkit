# 儿童绘本设计框架

## 核心框架：意外 → 成长

```
意外 → 松鼠经历 → 成长

不是「教」什么，是松鼠在经历中自然长出了什么
```

**为什么这个框架有效**：
- 小朋友跟着松鼠一起经历
- 不是被告知「要怎样怎样」
- 是在松鼠的故事里，自己长出了感受

---

## 两阶段创作流程

```
第一阶段：人机共同探索（定框架）
    ↓
第二阶段：调用相对固定的风格去执行（具体写作）
```

探索阶段是**开放**的，执行阶段是**固定**的。

**常见的跳坑**：
- 问「这本书能教小朋友什么」→ 这是执行阶段思维，要拉回探索阶段
- 直接定目标 → 先想森林里出了什么事

## 英语启蒙绘本设计原则

### 1. 核心词组设计原则

| 原则 | 说明 |
|---|---|
| 功能性 | 能在日常生活里用 |
| 重复性 | 同一个词组出现多次，场景不同 |
| 句型简单 | 短句，5-6岁能理解 |
| 情感连接 | 能表达情绪、请求、互动 |

### 2. 核心词组候选（松鼠系列）

| 核心词组 | 适用场景 |
|---|---|
| I found it! | 松鼠找到橡果，有成就感 |
| Can I help? | 帮邻居，社交礼貌 |
| Don't worry | 安抚朋友，情感支持 |
| Where is...? | 找东西/找路，探索好奇 |
| Look at me! | 展示自己，自我表达 |

### 3. 每本书的设计结构

```
核心词组 → 确定意外事件 → 围绕核心词组设计情节 → 松鼠自然说出词组 → 成长浮现
```

---

## 松鼠系列第一本书候选

### 方案A：Where is my home?
- **核心词组**：Where is...?
- **意外事件**：小兔子迷路，松鼠帮它找家
- **词组自然出现**：Where is my home? / Where is the river? / Where are you?
- **成长**：学会帮助别人 / 不害怕迷路，有人可以帮助

### 方案B：I found it!
- **核心词组**：I found it!
- **意外事件**：松鼠们的橡果被水冲走了，它们去找
- **词组自然出现**：I found it! / Look! / I see it!
- **成长**：遇到问题要想办法 / 一起合作能找到

### 方案C：Can I help?
- **核心词组**：Can I help?
- **意外事件**：大风吹断树枝，松鼠们一起修房子
- **词组自然出现**：Can I help? / Let me help! / I can help!
- **成长**：遇到困难可以请求帮助 / 帮助别人是开心的

---

## 选择标准

选哪个方案，取决于：
1. **核心词组的实用性**：这个词组孩子以后能用到吗？
2. **意外的戏剧性**：这个意外小朋友会觉得紧张/有趣吗？
3. **成长的自然度**：松鼠的成长是不是在这个经历里自然长出来的？

---

## 决策点

下一步需要和用户确认：
1. 选哪个核心词组？
2. 具体的意外事件是什么？
3. 松鼠在这个经历里会长出什么？
